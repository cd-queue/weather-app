# 🌦️ Weather App

A simple weather app built using HTML, CSS, and JavaScript.  
It shows real-time weather data using the OpenWeather API.

## 🚀 Features
- Search weather by city
- Temperature, humidity, and wind speed
- Weather conditions like Clouds, Rain, Mist, etc.
- Responsive design

## 🛠️ Technologies
- HTML
- CSS
- JavaScript
- OpenWeather API

## 📁 Project Structure

```bash
weather-app/
│
├── index.html
├── style.css
├── script.js
├── assets/
└── README.md
```

## ▶️ Run Project
1. clone the project
2. Open `index.html` in browser
